# The Federated Research Data Repository (FRDR)

Access to [FRDR](https://www.frdr-dfdr.ca/repo/) is free to any post-secondary researchers in Canada. FRDR provides curation and other services to support data deposit.

Lraries may support faculty interested in depositing to FRDR with the following resources:

-   Guide researchers in [setting up accounts](https://www.frdr-dfdr.ca/docs/en/before_depositing/)
-   Contact <support@frdr-dfdr.ca> for assistance
