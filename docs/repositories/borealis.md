# Borealis

The [Borealis data repository service](https://borealisdata.ca/) is provided through consortia. 

- COPPUL libraries wishing for a Borealis account must get it via [COPPUL](https://coppul.ca).
- BC ELN partner libraries that are not full or affiliate COPPUL members may join through [BC ELN](https://bceln.ca). 

# Support for Borealis 

BC ELN's only handles the consolidated invoicing for Borealis, for which no additional administrative fees or discounts are applied. All onboarding, training, operational support, and technical assistance are handled directly by Scholars Portal.
